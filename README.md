# 曲家崴生日祝福页

一个手机优先的生日祝福静态网页，包含：
- 主页面 `index.html`
- 一键分享页 `share.html`
- 本地头像与资料图素材

## 本地运行

```bash
python3 -m http.server 4782
```

然后访问：
- `http://127.0.0.1:4782/index.html`
- `http://127.0.0.1:4782/share.html`

## 部署

已提供 Render Blueprint 配置：`render.yaml`。
